Checking out the files, you should see that Ionic is using TypeScript by default.
We can remove TypeScript from the project by uninstalling some dependencies.
npm uninstall typescript
npm uninstall @types/jest
npm uninstall @typescript-eslint/eslint-plugin
npm uninstall @typescript-eslint/parser
npm uninstall @vue/cli-plugin-typescript
npm uninstall @vue/eslint-config-typescript
